# Vad är TypeScript?

- TypeScript är JavaScript fast med syntax för typer
- Starkt typat språk som transpilerar till JavaScript
- Utvecklat av Microsoft
- Open source, finns på Github: https://github.com/Microsoft/TypeScript (över 80k stars)
- Funnits sedan 2012


Notes: Typescript är JavaScript fast med syntax för typer plus enums och annat skoj. JavaScript i sig har ju en del primitiva typer, men inget som man deklarerar när man använder språket.
Det inferas av JavaScript runtimen, vilket kan vara browsern, node.js eller någon annan runtime.
Typescript däremot är ett starkt typat språk som transpilerar till JavaScript. Kommer berätta mer gällande en del typer som TypeScript har i senare slides.

---

### Vad innebär ett starkt typat språk?

- Strikta typ-regler i compile time

```jsx
function compat(arr) {
  if (orr.length > 10)
    return arr.trim(0, 10)
  return arr
}
```

Notes: Strikta typ-regler i compile time, ger oss fel och exceptions i compile time istället för runtime.
